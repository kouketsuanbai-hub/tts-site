## ステップ1：Cloudflare Pagesで公開する

**所要時間：5分（ブラウザ作業）**

1. https://dash.cloudflare.com にログイン
2. 左メニュー「Workers & Pages」→「Create」→「Pages」タブ
3. 「Connect to Git」→ GitHubアカウント（kouketsuanbai-hub）を連携
4. リポジトリ `kouketsuanbai-hub/tts-site` を選択
5. ビルド設定：
   - Framework preset: None
   - Build command: （空欄）
   - Build output directory: （空欄）
6. 「Save and Deploy」をクリック

URLが発行される（例：`tts-site-xxxx.pages.dev`）。

## ステップ3：Google AdSense申請

**所要時間：10分（申請のみ）／審査：数日〜数週間**

### 申請手順

1. https://www.google.com/adsense にアクセス
2. 「今すぐ申請」→ Googleアカウントでログイン
3. サイトのURL（Cloudflareで発行されたもの）を入力
4. 審査コード（`<script>`タグ）を受け取る
5. そのコードをindex.htmlの`<head>`内に貼る（Claudeでやる）
6. AdSenseの管理画面に戻り「審査をリクエスト」

### 審査通過後（Claude Codeでやる）

- index.htmlのツール下部に広告枠を1つ追加
- about.htmlの記事内に広告枠を1つ追加
- privacy.htmlにAdSense・Cookieに関する記述を追記

### 注意

- 審査落ちの主な理由：コンテンツ不足・ナビゲーション不備・ポリシー違反
- 1回落ちても再申請可能。記事を増やしてから再挑戦する手もある。
- 通過しても最初の収益は月0〜数百円が現実的。
